# Image Classifier

A simple image classification model trained using TensorFlow/Keras.

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Train model:
   python train.py

3. Predict:
   python predict.py --image test.jpg

## Model Files
- models/imageclassifiernewversion.h5
- models/imageclassifiernewversion.keras
